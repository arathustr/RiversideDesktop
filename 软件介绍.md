# Riverside 桌面客户端（Windows）——把 river-side.cc 用成“QQ”

这是一个面向 **river-side.cc（Discourse）** 的 Windows 桌面应用：把论坛的“聊天”和“帖子”合并进一个更像 QQ 的使用方式里——不需要反复开浏览器、切标签页，打开软件就能完成大多数日常操作。

> 适合人群：经常私聊/群聊、常逛“校友广场/二手交流”、想要更顺滑的发帖回帖体验的人。

---

## 你能用它做什么

### 1) 聊天（像 QQ 一样用）
- **会话列表 + 聊天窗口**：最近消息排序，常用会话可置顶。
- **支持公众频道**：例如“校友广场”“二手交流”等公共聊天频道。
- **支持私聊**：  
  - 点击头像可打开名片，并一键发起私聊  
  - **支持搜索用户 ID 直接发起聊天**（即使之前没聊过也能建立新会话）
- **消息更清晰**：聊天消息里显示用户信息（含 ID），更不容易认错人。
- **通知中心**：被回复/被 @ 提及（聊天 + 论坛），点击通知可直接跳转到对应楼层/消息。
- **图片与表情**：  
  - 输入框支持 **粘贴图片**直接发送  
  - 内置更丰富的表情（常用表情随手可发）
- **用户名片/空间**：点击头像可查看名片，并可进入对方空间（在应用内展示，不跳转网页）。

### 2) 发现（把“看帖回帖”做成更顺手的浏览方式）
- **大类浏览**：阅读帖子时不需要纠结细板块，按“大类”统一看就够了。
- **帖子在应用内打开**：  
  - 直接看帖、回帖、点赞、快捷评论  
  - 不再强制打开网页才能操作
- **发主题帖更稳**：发帖时会让你选择“具体板块”（论坛规则要求），避免发帖被拒绝（403）。

### 3) 更像“软件”的体验
- **可安装的正规安装包**：标准 Windows 安装流程（开始菜单/桌面快捷方式、卸载入口）。
- **更清爽的界面**：去掉多余说明文案，核心区域就是“聊天 / 发现”。
- **更少打扰**：登录后自动加载数据；需要时可手动刷新。

## 使用方法（第一次上手 30 秒）
1. 安装并打开软件  
2. 点击 **“打开登录窗口”**，在弹出的页面完成登录  
3. 回到软件：左侧选聊天会话或切到“发现”浏览帖子

## 常见问题

### Q1：为什么发帖要选“具体板块”？
论坛的分类结构里，“大类”用于统一浏览；但 **发主题必须指定可发帖的具体板块**。软件会在发帖窗口里提供选择，确保能发出去。

### Q2：会不会泄露账号信息？
软件通过官方站点的登录方式获取会话状态；日常操作在你的本机环境中完成。建议只从可信来源下载安装包。

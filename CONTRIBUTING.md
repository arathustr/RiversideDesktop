# Contributing

Thanks for helping improve Riverside Desktop!

## Development

Requirements: Node.js 20+ (recommended).

```bash
npm install
npm run typecheck
npm run dev
```

If PowerShell blocks `npm`, use `npm.cmd`.

## Pull requests

- Keep changes focused and easy to review
- Run `npm run typecheck` before submitting
- Avoid committing `node_modules/`, `dist/`, `release/` outputs

## Bug reports

Please include:
- What you did + what you expected
- Screenshots (if UI-related)
- App version + Windows version
- Any error shown in the app
